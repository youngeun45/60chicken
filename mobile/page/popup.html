팝업 호출

1. a태그를 사용하여 href에 대상 팝업의 id 지정

비밀번호 변경 권유 팝업
<a href="#popPwChange" class="button ui-pop-open">버튼</a>


2. 함수사용
popOpen(대상 팝업의 id)


<button class="button" onclick="popOpen('#popPwChange')">버튼</button>

팝업 닫기

1. 대상 팝업의 id 내에 .ui-pop-close 클래스의 버튼으로 닫기
<button class="button ui-pop-open">버튼</button>


2. popClose(대상 팝업의 id)
<button class="button" onclick="popClose('#popPwChange')">버튼</button>



일반 알럿 호출

getAlert(알럿메세지)


<button class="button" onclick="getAlert('사이드메뉴는 9개까지 구매가능 합니다')">버튼</button>
